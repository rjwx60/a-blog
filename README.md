## 一个利用Angular2设计构建的个人博客

* 此项目基于 Angular API 的“英雄编辑器” 案例 (https://angular.cn/docs) 的再设计和构建，能增删博客，并可根据喜好搜索能网易云的音乐歌单

* 项目基于 Angular-CLI 搭建，使用 Bootstrap 构建基本的页面



## 安装
	npm install

## 启动后台服务
	npm run mock

## 启动
	npm run start
